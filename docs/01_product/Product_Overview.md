# SuperSuit AI — 제품 개요

## 제품명

**SuperSuit AI**

## 제품 철학

> "울트론을 만드는 것이 아닌, 아이언맨 수트를 만드는 일"

AI가 구성원을 대체하는 것이 아니라, AI를 통해 한 명 한 명을 슈퍼히어로로 만들어 조직과 서비스를 변화시킨다. SuperSuit AI는 게임 개발 조직의 집단 지식을 체계적으로 축적하고, 구성원 누구나 그 지식을 즉시 활용할 수 있게 만드는 AI 기반 지식 관리 도구다.

## 핵심 문제

슈퍼센트와 같이 빠르게 게임을 반복 개발하는 조직에서는 다음 문제가 반복된다:

- 과거 게임에서 시도한 메카닉, 가설, 결과가 개인의 기억에만 남아 있다
- 새 게임을 기획할 때 "이런 메카닉, 전에 해봤나? 결과가 어땠나?"를 즉시 알 수 없다
- 성공/실패 패턴이 조직 자산으로 축적되지 않아 같은 실수가 반복된다
- 지표, 마케팅, 경쟁 리서치 데이터가 각 팀에 분산되어 통합 인사이트를 얻기 어렵다

## 핵심 기능

1. **다양한 데이터 업로드** — 회의 음성, 게임 지표, 마케팅 성과, 경쟁 리서치, A/B 테스트 결과를 업로드하면 자동으로 온톨로지에 축적
2. **온톨로지 그래프 시각화** — 게임을 중심으로 메카닉, 가설, 결과, 팀, 캠페인이 연결된 지식 그래프를 인터랙티브하게 시각화
3. **자연어 질문(LLM)** — "Snake Clash!에서 뱀 메카닉 성공 사례 알려줘" 같은 자연어 질문에 온톨로지 기반 인사이트 응답
4. **팀별 관리** — 기획/개발/아트/마케팅/QA 팀별로 관여 게임과 회의 기록 필터링

## 대상 사용자

**주 사용자: 게임 기획자 (PM)**
- "이런 메카닉, 전에 해봤나? 결과가 어땠나?"를 바로 알고 싶은 사람
- 새 게임 아이디어를 낼 때 과거 성공/실패 패턴을 즉시 참조하고 싶은 사람
- 인터뷰 근거: 바울 (Lead PD), 솔 (Junior PD), 네이선 (Data Analyst)

**부 사용자**
- 마케터 — 캠페인 성과 데이터를 게임 온톨로지와 연결해서 분석
- 데이터 분석가 — 지표 데이터를 온톨로지에 업로드하고 가설 검증 결과 기록
- 개발자 — 기술 메카닉 구현 현황과 업데이트 계획 확인

## 제품 포지셔닝

| 기존 방식 | SuperSuit AI |
|---|---|
| 개인 노션/문서에 기록 | 조직 공유 온톨로지에 자동 축적 |
| 시니어에게 직접 물어봄 | LLM에게 자연어로 질문 |
| 팀별 분산 데이터 | 게임 중심 통합 지식 그래프 |
| 회의 후 기록 소실 | 음성 업로드 → 자동 온톨로지 반영 |
